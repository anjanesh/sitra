2. Z Scores
========
( Zr ) Robust Z Score = ( Result - median ) / NIQR
NIQR - Nomalized inter quartile range = IQR x 0.7453
IQR = Q3 - Q1
Q1 = 1st quartile & Q3 = 3rd quartile


Zc = Classical Z Score = ( Result - Average ) / Standard Deviation
